# ⚽ Evaluación Final Módulo 4: Administrador de Usuarios

## 🎯 Objetivos de la Aplicación
Esta aplicación es una herramienta de gestión que consume una API externa (JSONPlaceholder) mediante el uso de `XMLHttpRequest`. El objetivo principal es aplicar los conceptos de Programación Orientada a Objetos (Clases y Métodos) en JavaScript para listar, buscar, filtrar y mostrar información detallada de una base de datos alojada en la nube, optimizando la interacción sin recargar la página.
