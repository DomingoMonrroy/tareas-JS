# Ministerio de las Mascotas - Registro Civil

## Objetivos de la aplicación
Este proyecto implementa un sistema de Registro Civil para el recién creado Ministerio de las Mascotas. La aplicación consta de dos partes principales:
1. **Backend (Node.js/Express):** Un servicio web que administra y almacena la información (nombre de la mascota y RUT del dueño) en formato JSON. Implementa todos los métodos requeridos (GET, POST, DELETE) para gestionar los registros.
2. **Frontend:** Una interfaz web funcional que se comunica con el servidor mediante Axios, permitiendo visualizar, buscar, registrar y eliminar mascotas. Además, incluye una capa visual para la captura y el manejo de errores.
